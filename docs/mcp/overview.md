# MCP Overview

## Objectif
Permettre a un agent IA de consulter l'etat vivant du systeme AURA sans lire uniquement du code statique.

## Priorites
- logs applicatifs
- schemas de base de donnees
- etat des jobs

## Principes
- exposition en lecture seule par defaut
- noms de ressources stables
- filtrage des secrets et donnees sensibles
