# MCP Use Cases

## Diagnostic d'un telechargement
- lire `jobs/active`
- ouvrir l'historique d'un job
- consulter les logs lies a son identifiant

## Verification d'un schema
- lire `db/schema/postgres`
- comparer les entites documentees avec les tables reelles

## Diagnostic player indirect
- verifier si des jobs ou providers echouent
- verifier si des metadonnees attendues manquent
